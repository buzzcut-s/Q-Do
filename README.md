## Getting Started
To get a local copy up and running follow these simple steps.

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/buzzcut-s/Q-Do.git
   ```
2. Build using
   ```sh
   cd Q-Do/
   mkdir build 
   cd build
   qmake ..
   make
   ```
3. Run using
   ```sh
   ./Q-Do
   ```
